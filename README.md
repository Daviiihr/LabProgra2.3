# LabProgra2.3
Este laboratorio tiene como objetivo aplicar los conceptos de **programación funcional y orientada a objetos** en Python.  
A lo largo de este trabajo se desarrollan ejercicios utilizando las funciones y estructuras más comunes de este paradigma, como:

- `yield`  
- `iter()`  
- **Listas por comprensión**  
- `lambda`  
- `map()`  
- `filter()`  
- `reduce()`
